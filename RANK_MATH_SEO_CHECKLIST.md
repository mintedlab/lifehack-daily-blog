## 💡 Rank Math 최적화 팁

### 콘텐츠 작성 시
1. **키워드 밀도**: 전체 콘텐츠의 1-2% (자연스럽게)
2. **제목에 키워드**: H1, H2에 주요 키워드 포함
3. **첫 10% 내 키워드**: 도입부에 주요 키워드 언급
4. **이미지 Alt 태그**: 모든 이미지에 설명 Alt 태그

### 기술적 SEO
1. **URL 구조**: 짧고 키워드 포함 (예: /posts/productivity-apps-2025)
2. **헤딩 구조**: H1 → H2 → H3 순서 준수
3. **Schema 마크업**: Article, BlogPosting 필수
4. **모바일 우선**: 모바일 UX 최우선

### 사용자 경험
1. **읽기 쉬운 구조**: 짧은 단락, 불릿 포인트
2. **비주얼 요소**: 이미지, 표, 인포그래픽
3. **내부 링크**: 관련 글 3-5개 링크
4. **로딩 속도**: 3초 이내 목표
